# AI Booking Chatbot

This is a full-stack AI-powered chatbot for booking flights. It includes:

- 🌐 **Frontend**: React  
- ⚙️ **Backend**: FastAPI  
- 🔐 **Auth**: Firebase (Phone Authentication)  
- 🤖 **LLM**: DeepSeek(configurable)

---
